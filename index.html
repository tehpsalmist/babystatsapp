About This Project

The App

The idea for this app came to mind while my family was exchanging conjectures about the stats of my pending 3rd child. This project came about very late in the game, also. With Mommy's due date having come and gone (July 18!), it wasn't the most obvious project to start into. But, Google's Firebase platform came up big in the clutch, making storage very easy to work with (and learn from scratch!). The outstanding documentation from Bootstrap (I used v4-alpha6 for fun) and KnockoutJS also made the road to completion very smooth and pleasant. I tried hard to follow best practices concerning separation of concerns in my Knockout MVVM framework, but I must confess this was a substandard utilization of Bootstrap's amazing UI options. 😜 Who knows, maybe I'll give it a facelift if the baby takes too long.

The Developer

My name is Ben Steward, and I am a student of the Web! At the time of writing (July 2017) I am finishing up Udacity's Front-End Web Developer Nanodegree and doing business development for a local web design company. I am also taking courses in Node.js, React, and Angular. You can check out the code behind this project in my GitHub repo, or if you're looking for a game to kill some time, try your hand at No, No, Abi!, a simple Frogger knock-off built on HTML5 Canvas (requires keyboard).
